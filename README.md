# Servo Motors Sweep

An Arduino project that controls four servo motors using the Servo library.

The motors perform a sweep motion for approximately two seconds and then stop at a fixed position of **90°**, meeting the required task specifications.

## Features

- Control four servo motors simultaneously
- Sweep all motors from 0° to 180°
- Return from 180° to 0°
- Hold all motors at 90°
- Built using the Arduino Servo library

---

## Technologies Used

- Arduino
- C++
- Servo Library
- Tinkercad

---

## How It Works

1. Attach four servo motors to the Arduino.
2. Initialize each servo using the Servo library.
3. Move all motors together from **0°** to **180°**.
4. Move them back from **180°** to **0°**.
5. Set all servo motors to **90°** and keep them in that position.
